🚀 GETTING STARTED (5 minutes)
==============================

STEP 1: Install Dependencies
  pip install -r requirements.txt

STEP 2: Install Playwright Browsers
  playwright install

STEP 3: Get Groq API Key
  Visit: https://console.groq.com
  Sign up (free)
  Copy your API key

STEP 4: Configure .env
  Edit .env file:
  GROQ_API_KEY=your_api_key_here

STEP 5: Run!
  python main.py interactive

That's it! 🎉


💻 USAGE EXAMPLES
=================

COMMAND LINE:
  # Interactive mode
  python main.py interactive

  # Execute task
  python main.py task "Search Google for Python"

  # Batch execution
  python main.py batch sample_tasks.json

  # Show configuration
  python main.py config --show

PYTHON API:
  import asyncio
  from browser_agent import BrowserAgent

  async def main():
      agent = BrowserAgent()
      result = await agent.execute_task(
          "Go to Google and search for LangChain"
      )
      print(result)

  asyncio.run(main())

ADVANCED:
  # With retry logic
  from advanced_config import TaskExecutor
  
  executor = TaskExecutor(agent)
  result = await executor.execute_with_retry(
      "Your task", max_retries=3
  )

  # Batch execution
  results = await executor.execute_batch([
      "Task 1",
      "Task 2", 
      "Task 3"
  ])
